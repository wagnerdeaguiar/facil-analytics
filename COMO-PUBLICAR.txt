O QUE VOCÊ PRECISA FAZER (3 coisas — o script explica na hora)
--------------------------------------------------------------------------------

1) NEON (banco de dados — grátis)
   - Criar projeto → copiar "Connection string"
   - Colar quando o script pedir

2) VERCEL (hospedagem — grátis)
   - Login com GitHub → importar "facil-analytics"
   - Colar as variáveis que o script mostrar em Settings → Environment Variables

3) GOOGLE (login dos usuários — obrigatório em produção)
   - console.cloud.google.com → Credenciais → OAuth
   - Colar Client ID e Secret no arquivo .env.vercel

Depois do primeiro deploy, importe o histórico Lotofácil:
   - Entre no site → Resultados → Importar planilha
   - Ou rode: npm run db:import-xlsx (com DATABASE_URL do Neon)
